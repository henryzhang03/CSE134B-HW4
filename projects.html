Projects

My very first project was a Top-Down Shooter Game where I programmed a top-down shooter game for AP Computer Science A final independent project (Java). I gained experience with Canvas, 2D Graphics, player inputs, and game physics, and implemented and learned about Data Structures and Object-Oriented Programming.

My second project was an Analysis of Micro and Macro Water Health Trends. We submitted this to HackUMBC and we developed hardware and software that measures and analyzes 4 components of water quality from households in local communities: pH, dissolved oxygen, turbidity, and specific conductance. We utilized EasyEDA for hardware and wrote Python scripts for data analysis (PyMongo) and designed an organized user interface for citizens to work with.
